# 📚 01. 필수 — pandas를 위한 Python 기초 요약

이 노트북은 **pandas를 배우기 전에 알아야 할 Python 기초 개념**을 다룹니다. 



---

## 핵심 개념 및 기술
- **리스트**: DataFrame의 각 컬럼은 리스트로 구성됨.
- **딕셔너리**: `pd.DataFrame(딕셔너리)`로 DataFrame 생성 가능.
- **for 루프**: `iterrows()`로 DataFrame의 행을 순회.
- **조건문**: boolean 인덱싱을 통해 조건에 맞는 데이터 필터링.



---

## 주요 코드 예제
- 리스트와 딕셔너리를 활용한 DataFrame 생성 및 접근 방법.
- 조건문을 사용한 데이터 필터링 예시.





## 핵심 포인트
1. 리스트는 DataFrame의 각 컬럼을 구성하는 기본 자료구조입니다.
2. 딕셔너리는 DataFrame 생성 시 컬럼명과 데이터를 연결하는 중요한 역할을 합니다.
3. for 루프와 조건문은 pandas 데이터 조작에 필수적인 도구입니다.
4. 이 노트북은 pandas 사용을 위한 기초 개념을 체계적으로 정리하고 있습니다.
5. 다음 단계에서는 함수, 슬라이싱, 인덱싱을 배워 더 심화된 pandas 활용법을 익힐 예정입니다.
